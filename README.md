## Installation

> [!CAUTION]
> Untested software. Be prepared for crashes, alarms that don't go off, missed readings.

Download and install the latest APK from the [release page](https://github.com/awowogei/Messina/releases)  
Remember to kill any other app you use to connect to the sensor.
